## PRD - 删除 Gate 机制

**需求来源**: 用户决策 - Gate 机制会让 AI 停下来征求意见，破坏 Stop Hook 的自动循环
**功能描述**: 从 engine 中完全移除 Gate 审核系统，只保留 Stop Hook + branch-protect + CI 三层防线
**涉及文件**:
- skills/gate/ 整个目录（删除）
- scripts/gate/ 整个目录（删除）
- skills/dev/SKILL.md（去掉 gate 引用）
- skills/dev/steps/01-prd.md（去掉 gate:prd 调用）
- skills/dev/steps/04-dod.md（去掉 gate:dod/qa 调用）
- skills/dev/steps/05-code.md（去掉 gate:audit 调用）
- skills/dev/steps/06-test.md（去掉 gate:test 调用）
- skills/dev/steps/07-quality.md（去掉 gate 引用）
- skills/dev/steps/10-learning.md（去掉 gate:learning 调用）
- .gate-*-passed 文件（删除）
- tests/gate/ 整个目录（删除）
- scripts/run-gate-tests.sh（删除）
- scripts/qa-with-gate.sh（删除）
- regression-contract.yaml（移除 gate 相关条目）
- features/feature-registry.yml（移除 gate 相关条目）
- CHANGELOG.md（记录变更）

**成功标准**:
1. skills/gate/ 目录不存在
2. scripts/gate/ 目录不存在
3. tests/gate/ 目录不存在
4. .gate-*-passed 文件不存在
5. /dev step 文件中无 gate 调用
6. SKILL.md 流程图无 gate 引用
7. CI 通过
8. 所有现有测试通过（gate 测试除外）

**非目标**:
- 不修改 branch-protect.sh（保留文件存在性检查）
- 不修改 stop.sh（保留循环控制）
- 不修改 CI 配置（CI 检查不变）
- 不删除 qa/audit skill 本身（只删 gate 调用）
