# PRD: 修复 CI 残留检查与 Release 证据冲突

## 背景

- L4 Evidence Check 需要 .dod.md 文件验证 release
- PRD/DoD Residue Check 拒绝 develop/main 分支的 .dod.md 文件
- 这两个检查产生冲突，导致 release PR 无法合并

## 修复方案

修改 PRD/DoD Residue Check：
- 如果 .layer2-evidence.md 存在，说明是 release 证据文件
- 此时允许 .dod.md 存在
- 否则继续原有的检查逻辑

## 验收标准

- [x] CI 残留检查在 release 证据存在时跳过
- [x] 正常 PRD/DoD 残留仍被检测
