# PRD - Hook 保护全局配置目录

**需求来源**: 用户发现 Claude 可以直接修改 `~/.claude/` 下的文件，跳过 /dev 工作流

**功能描述**:
1. 在 `branch-protect.sh` 中增加规则：阻止直接修改 `~/.claude/hooks/` 和 `~/.claude/skills/` 下的文件
2. 同步之前在 `~/.claude/` 做的修改（v11 PRD/DoD 检查、T1-T6 命名、develop 分支引用）
3. 建立正确的部署流程：zenithjoy-engine (develop → main) → ~/.claude/
