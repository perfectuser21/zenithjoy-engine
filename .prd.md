# PRD - 添加状态追踪脚本

## 需求来源

有头（交互式）和无头（Cecelia）模式都需要同步状态到 Core/Notion。

## 功能描述

添加 `track.sh` 脚本到 `/dev` Skill，在工作流关键点调用来同步状态：
- 任务开始时创建 run
- 每个步骤时更新进度
- 完成/失败时更新最终状态

## 成功标准

- `track.sh` 可执行且功能正常
- SKILL.md 包含追踪文档
- 部署到 ~/.claude/skills/dev/ 成功
