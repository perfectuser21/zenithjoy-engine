# PRD - GitHub Actions Auto Merge

Priority: P1

## 背景

已升级到 Team Organization (A+ 100% 保护)：
- ✅ required_approving_review_count: 1（必须人工审核）
- ✅ restrictions: 空（禁止直接 push）
- ✅ enforce_admins: true

现需要配置自动合并工作流，在 CI 通过 + 人工 approve 后自动合并 PR。

## 目标

配置 GitHub Actions 自动合并：
1. 监听 PR review 和 CI 完成事件
2. 检查条件：approved + CI success
3. 自动执行 squash merge

## 验收标准

- auto-merge.yml 工作流文件存在
- CI 通过后，用户 approve PR
- PR 自动合并到目标分支
- 无需人工点击"Merge"按钮
