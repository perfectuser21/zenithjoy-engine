# Task Checkpoint 强制执行

## 目标
在 /dev 流程中强制创建 Task Checkpoint，让用户看到进度

## 功能描述
1. Step 3 创建分支后，自动创建 11 个 Task（对应 Step 1-11）
2. .dev-mode 添加 tasks_created: true 字段
3. branch-protect.sh 检查 tasks_created 字段
4. 每个 Step 文件添加 TaskUpdate 指令

## 涉及文件
- skills/dev/steps/03-branch.md（添加 Task 创建）
- hooks/branch-protect.sh（添加 tasks_created 检查）
- skills/dev/steps/*.md（添加 TaskUpdate 指令）

## 成功标准
- [ ] Step 3 创建所有 Task
- [ ] .dev-mode 包含 tasks_created: true
- [ ] Hook 检查 tasks_created
- [ ] 用户能看到进度
