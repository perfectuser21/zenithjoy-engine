# PRD - v9.1.4 独立交叉验证

Priority: P1

## 背景

v9.1.3 实现了：
1. ✅ 可证伪（反证能 fail）
2. ✅ 可审计（explain 有分母来源 + 匹配原因）
3. ✅ 强制执行（CI 守门）

但还差：
4. ❌ 独立交叉验证（避免"自证"）

## 问题

1. **分母自证**：coverage:rci 的分母来自它自己的扫描规则，可能漏扫
2. **守门可撤**：有人可能悄悄移除 CI 中的 coverage:rci 检查

## 目标

从"自证"变成"交叉验证"，让 100% 覆盖率"放心到不再焦虑"。

## 交付物

1. `--stats` 模式：用独立方式统计分母，输出对比
2. `assert-ci-guards.sh`：防篡改哨兵，验证 CI 守门存在

## 成功标准

- `npm run coverage:rci -- --stats` 显示独立分母核对
- `bash scripts/devgate/assert-ci-guards.sh` 验证 CI 守门
- 两个数字能对上 → 分母可信
