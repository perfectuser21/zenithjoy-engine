# Credential Guard Hook

## 目标
添加 Claude Code hook 拦截代码中的真实凭据

## 功能
- 检测 Write/Edit 操作中的敏感 token 模式
- 自动拦截并提示正确做法
- 跳过 ~/.credentials/ 目录

## 验收标准
- [ ] hook 能识别 Notion/GitHub/OpenAI 等 token
- [ ] 占位符 (YOUR_XXX) 不会被拦截
- [ ] ~/.credentials/ 写入正常
